Basic CSS size/units

What is a CSS Unit

A CSS unit determines the size of a property youâ€™re setting for an element or its content. For example, if you wanted to set the property margin of a paragraph, you would give it a specific value. This value includes the CSS unit. In ths case, margin is the property, 20px is the value, and px (or "pixel") is the CSS unit.

Here are some basic CSS unit

  • pc(Picas):

    A pica is equivalent to 12 points; thus, there are 6 picas per inch.

  • pt(Points):

    It represents 1/72 of an inch. For example, 12pt font size traditionally means characters are 1/6 inch or 12 points tall.

  • px(Pixels):

    Pixels (px) are relative to the viewing device. For low-dpi devices, 1px is one device pixel (dot) of the display. For printers and high resolution screens 1px implies multiple device pixels. Pixels (px) are typically the most popular absolute unit for screens. Centimeters, millimeters, and inches are more common for print and you may not have even known they were options!

  • em:

    Relative to font size of the parent, in the case of typographical properties like font-size, and font size of the element itself, in the case of other properties like width.

  • rem:

    Relative to font size of the root element.

  • %(Percentage):

    A Percent represents a fraction of some other value. For example, 50%. Percentage values are always relative to another quantity. For example, an element's length is relative to its parent element's length.

  • vh:

    Relative to 1% of the viewport's height.
